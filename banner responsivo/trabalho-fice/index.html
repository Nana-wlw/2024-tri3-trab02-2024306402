E.YCall - PROJETO PARA A AUTOMATIZAÇÃO DE CHAMADAS ESCOLARES

1 autoria, Edson Eduardo dos Santos - 2 autoria, Yasmin Andrade Correia - 3 autoria, Joice Seleme Mota

INTRODUÇÃO

A presença dos alunos é crucial para o sucesso educacional, mas o processo tradicional de chamada consome muito tempo em sala de aula. Pesquisas mostram que até 39% do tempo da aula é perdido com atividades como chamada e organização. No Instituto Federal Catarinense, essa preocupação foi identificada, especialmente em turmas maiores. Para resolver o problema, os estudantes Edson e Yasmin desenvolveram um projeto usando biometria digital, considerada a melhor opção pela sua precisão e eficiência, para automatizar o controle de presença e tornar o ambiente educacional mais produtivo.

OBJETIVOS

O objetivo deste projeto é automatizar o controle de presença escolar substituindo o processo manual da chamada por um sistema mais eficiente, utilizando biometria digital. A solução visa otimizar o tempo em sala de aula, aumentar a produtividade e garantir uma identificação precisa e segura dos alunos. Além disso o projeto busca criar um protótipo que integra a leitura de digitais armazenamento de dados e envio de informações via wi-fi, proporcionando uma alternativa prática e acessível para melhorar a gestão de frequência escolar.

METODOLOGIA

Este projeto é uma releitura de um sistema de chamadas automáticas usando Arduino, substituindo cartões por Leitura digital e adicionando um módulo WIFI ESP8266 para maior eficiência. O processo incluiu pesquisa técnica, testes com componentes, e ajustes no hardware e como a troca da placa Arduino Mega2560 pela UNOr3 devido a interferências. A integração entre o leitor biométrico e o display foi o maior desafio, exigindo múltiplos testes e ajustes. O sistema utiliza C++ com JSON para armazenar biometrias, e será validado com testes online no Tinkercad antes de testes práticos finais.

RESULTADO

Atualmente, o projeto está em fase de implementação. O sistema inclui um leitor biométrico para registro de digitais, um display para confirmar a presença dos alunos e um banco de dados. O protótipo já funciona com o leitor e display em testes sendo capaz de cadastrar e verificar digitais(Figura 1). A próxima etapa será enviar digitais para um servidor ou um banco de dados gerando um arquivo de chamada para o professor.

CONCLUSÕES

O E.YCall é um sistema que usa biometria digital para otimizar o controle de presença escolar agilizando o processo e tornando o mais eficiente. O projeto foi desenvolvido com leitores biométricos, armazenamento de dados e testes em plataformas como Tinkercad. Além de otimizar a gestão de presença, o E.YCall pode ser aplicado em outros ambientes. O projeto recebeu apoio do edital 12/2024 - IFMaker com uma bolsa de pesquisa para alunos do IFC campus Camboriú.
